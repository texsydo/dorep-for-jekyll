# Jekyll Site

Copy the following user files to the root of the Jekyll project before building
the Jekyll site:

**Mandatory**:

- `_data/site_meta.yml`
- `_data/footer.yml`
- `favicon.png`
- `favicon.ico`

**Recommended**:

- `README.md` of the user project.
- `LICENSE` of the user project (there can be many).
